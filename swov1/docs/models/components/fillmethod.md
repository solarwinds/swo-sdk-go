# FillMethod

Method for filling missing data points in the range.


## Values

| Name                      | Value                     |
| ------------------------- | ------------------------- |
| `FillMethodNone`          | NONE                      |
| `FillMethodZeroFill`      | ZERO_FILL                 |
| `FillMethodNullFill`      | NULL_FILL                 |
| `FillMethodLastValueFill` | LAST_VALUE_FILL           |
